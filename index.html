Коротка інформація про ОС GNU/Linux

GNU/Linux — це операційна система, яка поєднує ядро Linux і програмне забезпечення проекту GNU. Це одна з найпопулярніших систем з відкритим кодом, яка використовується на серверах, комп'ютерах, мобільних пристроях та вбудованих системах.

Приклади команд роботи в Linux CLI

  1. pwd — показує поточний каталог.
  2. echo "текст" — виводить текст.
  3. ls — виводить список файлів і директорій у каталозі.
  4. cd <каталог> — змінює поточний каталог.
  5. touch <файл> — створює новий файл.
  6. mkdir <каталог> — створює новий каталог.
  7. cp <файл> <каталог> — копіює файл у каталог.
  8. mv <файл> <каталог> — переміщує або перейменовує файл.
  9. rm <файл> — видаляє файл.
  10. history — виводить історію команд.

BASH-скрипт, що імітує авторизацію в системі


#!/bin/bash
echo "Введіть логін:"
read login
echo "Введіть пароль:"
read -s password

if [[ $login == "admin" && $password == "1234" ]]; then
    echo "Авторизація успішна!"
else
    echo "Невірний логін або пароль!"
fi
